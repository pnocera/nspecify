# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-15

### Added
- Initial project setup
- Core CLI architecture with Commander.js
- Support for multiple AI assistants (Claude Code, Gemini CLI, GitHub Copilot, Cursor)
- Cross-platform support (POSIX Shell and PowerShell scripts)
- Template download functionality from GitHub releases
- Interactive prompts for project initialization
- Git integration for checking repository status
- Comprehensive error handling and user-friendly messages
